# Canvas Checkerboard Example

This example demonstrates how to render a complex `Graphic` onto an HTML5 Canvas using MoonBit's WebAssembly backend and the `@canvas` package.

## Features

- **WebAssembly Integration**: Shows how to use MoonBit to drive standard web technologies like the `<canvas>` element.
- **Canvas Context**: Demonstrates the use of `@canvas.CanvasCtx` to translate `Graphic` elements into canvas drawing commands.
- **Dynamic Fitting**: Shows how to use `.fit_to()` to automatically scale and center a graphic within the available canvas area.
- **Interaction**: Provides a foundation for building interactive graphics in the browser.

## How to Run

To run this example, execute the following command from the root of this repository:

```bash
./run-canvas-checkerboard.sh
```
